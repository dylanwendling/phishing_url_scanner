# phishing_url_scanner
A desktop URL Threat Scanner that integrates with the VirusTotal API to analyze and score URLs for malicious threats. Features real-time scanning, blacklist management, scan history logging. Built with a layered architecture including backend API endpoints, data repositories, and cross-cutting security and error handling.
